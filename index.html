Tags basicas

Tag Descrição Exemplo
<!DOCTYPE> Define o tipo de documento <!DOCTYPE html>
<html> Define o início de um documento HTML <html> ... </html>
<head> Contém informações sobre o documento
(metadados, título da página, links, scripts, etc.).
Não inclui conteúdo das páginas		 <head>
          <meta charset="UTF-8">
          <title>site HTML<title/>
</head>
<title> Define o título da página, que aparece na aba do navegador <title>site HTML</title>
<body> Contém o conteúdo visível da página, é o "corpo da página" <body>
          <h1>Site sobre tags<h1/>
          <p>HTML<p/>
</body>
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> Definem os títulos (h1 é o maior e h6 o menor) <h1>Título 1</h1>
<p> Define um parágrafo <p>Texto do parágrafo</p>
<br> Insere uma quebra de linha (pula uma linha). É uma tag self close <br> ou <br/>
<hr> Insere uma linha horizontal (separador). É uma tag self close <hr> ou <hr/>
<em> ou <i> Define texto em itálico, geralmente usado para destacar palavras de forma sutil <em>Texto em itálico</em>
<strong> Define texto em negrito, geralmente usado para enfatizar palavras de forma mais forte <strong>Texto em negrito</strong>
<q> Define uma citação curta, exibida entre aspas <q>Isto é uma citação curta</q>
<pre> Exibe texto pré-formatado, mantendo espaçamento e quebras de linha (monstra o texto sem nenhuma formatação) <pre>Texto
Pré-formatado</pre>

Formularios

Tag Descrição Exemplo
<form> Define o início de um formulário. A tag deve conter os campos de entrada e o botão para envio. <form> ... </form>
<input> Define um campo de entrada. Pode ser usado para vários tipos de dados (texto, senha, botão, etc.). É uma tag self close <input type="text" name="nome">
<label> Define um rótulo para um elemento de entrada de dados em um formulário. <label for="nome">Nome:</label>
<textarea> Define uma área de texto multilinha onde o usuário pode digitar um texto longo. <textarea name="comentario" rows="10" cols="50">Digite seu comentário</textarea>
<select> Define uma lista suspensa (menu de seleção) com várias opções. <select name="opcao">
          <option value="1">Opção 1</option>
          <option value="2">Opção 2</option>
</select>
<option> Define uma opção dentro de um elemento. <option value="1">Opção 1</option>
<button> Define um botão que pode ser usado para enviar um formulário ou executar uma ação específica. <button type="button">Enviar</button>
<fieldset> Define um conjunto de elementos de formulário agrupados, normalmente usado para criar uma área com borda (agrupa elementos de formulários). <fieldset> ... </fieldset>
<legend> Define um título para um <fieldset>, fornecendo um contexto para o grupo de campos. <legend>Informações de contato</legend>
<input type="submit"> Define um botão de envio para o formulário. <input type="submit" value="Enviar">
<input type="radio"> Define um botão de opção (radio button) que permite escolher uma única opção entre várias. <input type="radio" name="genero" value="masculino">Masculino</input>
<input type="checkbox"> Define uma caixa de seleção (checkbox), onde o usuário pode selecionar várias opções. <input type="checkbox" name="interesses" value="esportes"> Esportes</input>
<input type="email"> Define um campo para inserção de um e-mail, com validação automática do formato. <input type="email" name="email">
<input type="password"> Define um campo de entrada para senha, com o texto oculto. <input type="password" name="senha">
<input type="number"> Define um campo numérico, com validação de entrada de números. <input type="number" name="idade">
<input type="date"> Define um campo de entrada de data. <input type="date" name="data">
<datalist> Define uma lista de opções pré-definidas para um campo de entrada. <input list="cidades" name="cidade">
<datalist id="cidades">
   <option value="São Paulo">
   <option value="Rio de Janeiro">
</datalist>
<output> Exibe o resultado de um cálculo ou ação (realiza a saida de dados). <output name="resultado">0</output>

Elementos de midia

Tag Descrição Exemplo
<img> Insere uma imagem na página. É uma tag self close. <img src="nomeDaImagemQueEstaSalvaNoSeuPc.extensão ou linkDaImagemNaWeb" alt="Descrição da imagem">
<audio> Define um arquivo de áudio, permitindo que o usuário reproduza sons diretamente na página. <audio controls>
          <source src="audio.mp3" type="audio/mp3">
</audio>
<video> Define um arquivo de vídeo, permitindo que o usuário reproduza vídeos diretamente na página. <video controls>
          <source src="video.mp4" type="video/mp4">
</video>
<source> Especifica múltiplos arquivos de mídia para o elemento <audio> ou <video> e é utilizado dentro dessas tags. É uma tag self close. <source src="video.mp4" type="video/mp4">
<iframe> Insere um conteúdo de outro documento HTML dentro da página atual, como um vídeo do YouTube ou mapa do Google. <iframe src="https://www.youtube.com/embed/xyz"></iframe>
<embed> Insere um conteúdo externo interativo ou multimídia, como flash ou arquivos PDF. É uma tag self close. Pode ser usado para carregar conteúdos que não são nativamente suportados diretamente em HTML. <embed src="arquivo.swf" width="300" height="200">
<object> Define um objeto de mídia ou um recurso externo, como um arquivo Flash, imagem ou applet Java. Permite não apenas a exibição de conteúdo, mas também a interação com objetos externos. <object data="documento.pdf" type="application/pdf"></object>
<picture> Permite definir diferentes imagens para diferentes tamanhos de tela ou resoluções. <picture>
          <source srcset="imagem-800w.jpg" media="(max-width: 800px)">
          <img src="imagem-default.jpg" alt="Imagem">
</picture>
<track> Especifica legendas ou outras faixas de texto para o elemento <video> ou <audio>. <video controls>
          <track src="legenda_en.vtt" kind="subtitles" srclang="en" label="English">
</video>
<figure> Representa um conteúdo autônomo, como uma imagem, diagrama ou gráfico, comumente utilizado junto com <figcaption> para legendas. <figure>
   <img src="imagem.jpg" alt="Descrição da imagem">
   <figcaption>Legenda da imagem</figcaption>
</figure>
<figcaption> Fornece uma legenda ou descrição para o conteúdo dentro de um elemento <figure>. <figcaption>Legenda da imagem</figcaption>

Links

Tag Descrição Exemplo
<a> Define um link (âncora) que leva a outra página ou a um destino dentro da mesma página. <a href="https://www.exemplo.com">Clique aqui</a>
<link> Define um link externo para arquivos como folhas de estilo (CSS), ícones, etc. É uma tag self close. <link rel="stylesheet" href="estilos.css">
<base> Define a base para os links na página, ou seja, um caminho padrão para as URLs(todas as URLs vão ser interpretadas como se começassem com o link da tag base). É uma tag self close. <base href="https://www.exemplo.com/">
<link rel="stylesheet" href="estilos.css">(O site va interpretar o link "estilos.css" como "https://www.exemplo.com/estilos.css")
<area> Define uma área clicável dentro de uma imagem com o uso da tag <map>. <map name="mapa">
          <area shape="rect" coords="34,44,270,350" href="https://www.exemplo.com" alt="Exemplo" target="_blank">
</map>
<nav> Define um bloco de navegação (menu) para facilitar a navegação entre as páginas(agrupa elementos relacionados a links). <nav>
          <a href="#home">Home</a>
          <a href="#sobre">Sobre</a>
</nav>
<button> Define um botão que pode ser usado para navegar entre páginas (com JavaScript). <button onclick="window.location.href='https://www.exemplo.com'">Visitar Exemplo</button>

Listas

Tag Descrição Exemplo
<ul> Define uma lista não ordenada (com marcadores). <ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
<ol> Define uma lista ordenada (com números, por padrão). <ol>
    <li>Primeiro</li>
    <li>Segundo</li>
</ol>
<li> Define um item dentro de uma lista (pode ser usado dentro de <ul> ou <ol>). <ul>
    <li>Exemplo de item</li>
</ul>
<dl> Define uma lista de descrição, que agrupa termos e suas definições. <dl>
    <dt>Termo 1</dt>
    <dd>Descrição do termo 1</dd>
</dl>
<dt> Define um termo dentro de uma lista de descrição (<dl>). <dl>
    <dt>Exemplo de termo</dt>
</dl>
<dd> Define a descrição de um termo em uma lista de descrição (<dl>). <dl>
    <dt>Termo</dt>
    <dd>Descrição do termo</dd>
</dl>

Tabelas

Tag Descrição Exemplo
<table> Define uma tabela para organizar dados em linhas e colunas. <table>
    <tr>
        <td>Dado 1</td>
        <td>Dado 2</td>
    </tr>
</table>
<tr> Define uma linha de tabela que contém células (<td> ou <th>). <table>
    <tr>
        <td>Dado 1</td>
        <td>Dado 2</td>
    </tr>
</table>
<td> Define uma célula de dados em uma tabela. Deve ser usada dentro de <tr>. <tr>
    <td>Dado 1</td>
    <td>Dado 2</td>
</tr>
<th> Define uma célula de cabeçalho em uma tabela. Geralmente usada para títulos de colunas ou linhas. <tr>
    <th>Cabeçalho 1</th>
    <th>Cabeçalho 2</th>
</tr>
<thead> Define o cabeçalho de uma tabela, agrupando linhas de cabeçalho (<tr>). <table>
    <thead>
        <tr>
            <th>Coluna 1</th>
            <th>Coluna 2</th>
        </tr>
    </thead>
</table>
<tbody> Define o corpo de uma tabela, agrupando as linhas de dados. <table>
    <tbody>
        <tr>
            <td>Dado 1</td>
            <td>Dado 2</td>
        </tr>
    </tbody>
</table>
<tfoot> Define o rodapé de uma tabela, agrupando linhas que geralmente contêm informações resumidas ou totais. <table>
    <tfoot>
        <tr>
            <td>Total</td>
            <td>100</td>
        </tr>
    </tfoot>
</table>
<caption> Fornece um título ou legenda para a tabela, geralmente exibido acima dela. <table>
    <caption>Tabela de Exemplo</caption>
    <tr>
        <td>Dado 1</td>
        <td>Dado 2</td>
    </tr>
</table>

Elementos de Agrupamento

Tag Descrição Exemplo
<div> Define um contêiner genérico para agrupar outros elementos ou conteúdo. Pode ser estilizado com CSS. <div>
    <p>Conteúdo dentro do div</p>
</div>
<section> Define uma seção de um documento, geralmente usada para agrupar conteúdo relacionado. <section>
    <h1>Título da Seção</h1>
    <p>Conteúdo da seção</p>
</section>
<article> Define um conteúdo independente e autocontido, como um artigo ou uma entrada de blog. <article>
    <h1>Título do Artigo</h1>
    <p>Conteúdo do artigo</p>
</article>
<header> Define o cabeçalho de um documento ou uma seção. Pode conter logotipos, menus de navegação e títulos. Ele pode ser usado apenas uma vez no codigo. <header>
    <h1>Cabeçalho</h1>
    <nav><a href="#home">Home</a></nav>
</header>
<footer> Define o rodapé de um documento ou uma seção. Pode conter informações como direitos autorais ou links adicionais. Ele pode ser usado apenas uma vez no codigo. <footer>
    <p>© 2024 Sua Empresa</p>
</footer>
<aside> Define um conteúdo complementar ou relacionado ao conteúdo principal, como barras laterais. <aside>
    <p>Conteúdo adicional</p>
</aside>
<nav> Define uma seção de navegação para links principais do site, como menus. <nav>
    <a href="#inicio">Início</a>
    <a href="#sobre">Sobre</a>
    <a href="#contato">Contato</a>
</nav>
<fieldset> Define um grupo de elementos dentro de um formulário, com uma borda ao redor. <fieldset>
    <legend>Informações Pessoais</legend>
    <label>Nome:</label>
    <input type="text" name="nome">
</fieldset>
<blockquote> Define uma citação em bloco, geralmente usada para destacar citações longas de outros autores. <blockquote>
    <p>Esta é uma citação importante</p>
</blockquote>
<span> Define uma seção de linha em um documento, usada para agrupar elementos inline(elementos que ocupam apenas uma linha). <span>Texto dentro de uma tag span</span>
<main> Define o conteúdo principal. Assim como o footer e o header ele pode ser usado apenas uma vez no codigo e normalmente é usado entre essas tags. <main>
    <h1>Título Principal</h1>
    <p>Conteúdo principal do site</p>
</main>
<details> Permitindo ao usuário visualizar ou ocultar conteúdo adicional pressionado uma seta. <details>
    <summary>Clique para ver mais</summary>
    <p>Conteúdo oculto inicialmente</p>
</details>
